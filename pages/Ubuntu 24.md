# Ubuntu 24.04
- ## Descripción
  Ubuntu 24.04, también conocido como Noble Numbat, es una de las distribuciones más populares de Linux, conocida por su facilidad de uso y su robustez. Es ideal tanto para principiantes como para usuarios avanzados, y es ampliamente utilizado en entornos de desarrollo y servidores. Ubuntu 24.04 continúa la tradición de ofrecer una experiencia de usuario fluida y una gran cantidad de software preinstalado y disponible a través de sus repositorios.
- ## Tips para Ubuntu 24.04
  1. **Actualizaciones Regulares**: Mantén tu sistema actualizado ejecutando regularmente `sudo apt update && sudo apt upgrade` en la terminal.
  2. **Uso de Workspaces**: Ubuntu permite el uso de múltiples workspaces para organizar mejor tus ventanas. Puedes cambiar entre ellos usando `Super + PageUp` y `Super + PageDown`.
  3. **Instalación de Software**: Utiliza el Centro de Software de Ubuntu para instalar aplicaciones de manera gráfica o la terminal para mayor control.
  4. **Personalización con GNOME Tweaks**: Instala GNOME Tweaks para personalizar la apariencia y el comportamiento de tu escritorio.
  5. **Uso de Snap y Flatpak**: Además de los paquetes tradicionales `.deb`, Ubuntu soporta Snap y Flatpak para instalar aplicaciones.

- ## Comandos sudo para Actualizaciones y Tareas Principales
- ### Actualizaciones
  
  | Comando | Descripción |
  |---------|-------------|
  | `sudo apt update` | Actualiza la lista de paquetes disponibles y sus versiones. |
  | `sudo apt upgrade` | Actualiza todos los paquetes instalados a sus últimas versiones disponibles. |
  | `sudo apt full-upgrade` | Realiza una actualización completa, incluyendo la eliminación de paquetes obsoletos si es necesario. |
  | `sudo apt dist-upgrade` | Similar a `full-upgrade`, pero puede manejar cambios más complejos en las dependencias. |
  | `sudo apt autoremove` | Elimina paquetes que ya no son necesarios, liberando espacio en el disco. |
  | `sudo apt autoclean` | Elimina los archivos de paquetes `.deb` descargados que ya no son necesarios. |
  | `sudo apt clean` | Elimina todos los archivos de paquetes `.deb` descargados, liberando espacio en el disco. |
  | `sudo do-release-upgrade` | Actualiza el sistema a la próxima versión de Ubuntu, si está disponible. |
- ### Tareas Principales
  
  | Comando | Descripción |
  |---------|-------------|
  | `sudo apt install <paquete>` | Instala un nuevo paquete. |
  | `sudo apt remove <paquete>` | Elimina un paquete instalado. |
  | `sudo apt purge <paquete>` | Elimina un paquete y sus archivos de configuración. |
  | `sudo apt search <término>` | Busca paquetes disponibles que coincidan con el término de búsqueda. |
  | `sudo apt show <paquete>` | Muestra información detallada sobre un paquete específico. |
  | `sudo systemctl start <servicio>` | Inicia un servicio del sistema. |
  | `sudo systemctl stop <servicio>` | Detiene un servicio del sistema. |
  | `sudo systemctl restart <servicio>` | Reinicia un servicio del sistema. |
  | `sudo systemctl enable <servicio>` | Habilita un servicio para que se inicie automáticamente al arrancar el sistema. |
  | `sudo systemctl disable <servicio>` | Deshabilita un servicio para que no se inicie automáticamente al arrancar el sistema. |
  | `sudo systemctl status <servicio>` | Muestra el estado de un servicio del sistema. |
  | `sudo reboot` | Reinicia el sistema. |
  | `sudo shutdown -h now` | Apaga el sistema inmediatamente. |
  | `sudo adduser <usuario>` | Añade un nuevo usuario al sistema. |
  | `sudo deluser <usuario>` | Elimina un usuario del sistema. |
  | `sudo usermod -aG sudo <usuario>` | Añade un usuario al grupo `sudo`, otorgándole privilegios de administrador. |
  | `sudo passwd <usuario>` | Cambia la contraseña de un usuario. |
  | `sudo nano /etc/fstab` | Edita el archivo `fstab` para configurar montajes de sistemas de archivos. |
  | `sudo mount /dev/sdXn /ruta/de/montaje` | Monta un dispositivo en una ruta específica. |
  | `sudo umount /ruta/de/montaje` | Desmonta un dispositivo montado en una ruta específica. |
  | `sudo chown <usuario>:<grupo> <archivo>` | Cambia el propietario y el grupo de un archivo o directorio. |
  | `sudo chmod <permisos> <archivo>` | Cambia los permisos de un archivo o directorio. |
  | `sudo ufw enable` | Habilita el firewall UFW (Uncomplicated Firewall). |
  | `sudo ufw disable` | Deshabilita el firewall UFW. |
  | `sudo ufw allow <puerto>` | Permite el tráfico en un puerto específico a través del firewall UFW. |
  | `sudo ufw deny <puerto>` | Bloquea el tráfico en un puerto específico a través del firewall UFW. |
  | `sudo ufw status` | Muestra el estado actual del firewall UFW y las reglas configuradas. |
  | `sudo journalctl` | Muestra los registros del sistema recopilados por `systemd-journald`. |
  | `sudo journalctl -u <servicio>` | Muestra los registros específicos de un servicio del sistema. |
  | `sudo dmesg` | Muestra los mensajes del kernel, útil para diagnosticar problemas de hardware. |
  | `sudo lshw` | Lista el hardware del sistema en detalle. |
  | `sudo dmidecode` | Muestra información detallada sobre el hardware del sistema desde la tabla DMI (SMBIOS). |
  | `sudo fdisk -l` | Lista las particiones de disco y sus detalles. |
  | `sudo mkfs.ext4 /dev/sdXn` | Formatea una partición con el sistema de archivos ext4. |
